# Автоматизированная DevOps-платформа CI/CD для веб-приложения

## Описание проекта

Данный проект представляет собой учебную автоматизированную DevOps-платформу CI/CD для веб-приложения, реализованную на примере FastAPI-сервиса.

Полная тема проекта:

**«Создание автоматизированной платформы для непрерывного тестирования и верификации стабильности программного обеспечения на примере веб-приложения с использованием контейнеризации и CI/CD».**

## Назначение проекта

Платформа предназначена для автоматизации полного цикла поставки приложения:

- непрерывного тестирования;
- контейнеризированной сборки;
- публикации артефактов;
- автоматического развертывания;
- проверки стабильности после поставки;
- сопровождения технической документации.

## Цель проекта

Разработать автоматизированную DevOps-платформу CI/CD для веб-приложения, обеспечивающую непрерывное тестирование, контейнеризированную сборку, автоматическое развертывание и верификацию стабильности программного обеспечения после поставки.

## Реализованные возможности

- запуск тестов через GitLab CI/CD;
- сборка Docker image;
- публикация image в GitLab Container Registry;
- автоматический deploy через shell runner;
- post-deploy health check;
- автоматическое обновление CHANGELOG;
- автоматическое обновление Wiki.
